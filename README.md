# Full-Stack TypeScript Portfolio Project

This is a foundational full-stack typed project using React (Vite) and Node.js (Express).

## Project Structure

- `/frontend`: React application built with TypeScript and Vite. It utilizes Tailwind CSS, standard routing, and framer-motion for UI interactions.
- `/backend`: Node.js Express server built with TypeScript, featuring a MongoDB connection setup (Mongoose).

## Getting Started

### 1. Backend Setup
1. Navigate to the backend directory: `cd backend`
2. Install dependencies: `npm install`
3. Copy the `.example-env` into `.env` (if applicable) and configure your variables like `PORT` and `MONGO_URI`.
4. Start the development server: `npm run dev`
   - The backend runs on `http://localhost:3001`

### 2. Frontend Setup
1. Navigate to the frontend directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
   - The frontend runs on `http://localhost:5173`

You can now start building out the REST API endpoints in the backend and the portfolio user interface in the React frontend!
